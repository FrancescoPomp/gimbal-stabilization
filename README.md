# gimbal-stabilization
Creating a 2-axis, IMU-stabilized gimbal. Sensor fusion and PID control will be done on a Teensy 4.0 microcontroller with BMI270 IMU, built as a controls characterization testbed in order to test PID performance and sensor fusion accuracy.
Microcontroller: Teensy 4.0
IMU: Sparkfun 6DOF BMI270 Breakout Board
